Color Tokens

All colors are defined as CSS custom properties on :root. Never use raw hex values in HTML or CSS — always reference the token.

Brand Colors

Primary Dark --brown #2C4048
Terracotta --terra #99461D
Terra Tint --terra-light #F0EDE8
Warm Gold --gold #D99F3D
Warm Brown reserved --green #7D5C50

On Dark --light #FAF7F2
Color Usage Rules
TokenPrimary UseDo not use for
--brownNav, footer, dark sections, dark CTAsBody text, inline links
--terraLinks, section labels, filled CTA button, card linksLarge background fills (too saturated)
--goldAccents on dark backgrounds onlyText on light backgrounds (fails contrast)
--bgPage body backgroundCard/panel surfaces
--surfaceCards, form inputs, white sectionsPage body (too stark)
--panelAlternating section backgrounds (warm stone)Text
--lightBody text on dark (--brown) backgroundsText on light surfaces
--mutedSupporting copy, metadata, captionsHeadings, CTAs

Buttons

All buttons use the base .btn class plus a variant. Buttons are pill-shaped (border-radius: 100px). Use .btn-lg for hero and CTA contexts.

.btn .btn-primary — Primary action on light backgrounds
Donate to NWCF Donate to NWCF
<a href="#" class="btn btn-primary">Donate to NWCF</a> <a href="#" class="btn btn-primary btn-lg">Donate to NWCF</a>
.btn .btn-secondary — Secondary / ghost on light backgrounds
Learn About Our Work Learn About Our Work
<a href="#" class="btn btn-secondary">Learn About Our Work</a>
.btn .btn-filled — Filled terracotta. Use ONLY for the primary Donate CTA
Donate to NWCF
<!-- Only used in .cta-band on index.html --> <a href="#" class="btn btn-filled btn-lg">Donate to NWCF</a>
.btn .btn-light — Outline on dark backgrounds
Learn More Learn More
<!-- Use on dark sections (.feature-section, .cta-band) --> <a href="#" class="btn btn-light">Learn More</a>
.btn-group — Button pair layout
Donate to NWCF Learn About Our Work
<div class="btn-group"> <a href="#" class="btn btn-primary btn-lg">Donate to NWCF</a> <a href="#" class="btn btn-secondary btn-lg">Learn About Our Work</a> </div>
Button Decision Matrix
ContextUse
Primary CTA on light bg (hero, events, research).btn .btn-primary
Secondary/ghost action on light bg.btn .btn-secondary
Donate CTA in CTA Band (dark section).btn .btn-filled
Secondary action on dark section.btn .btn-light
Donate pill in nav header.nav-donate a (auto-styled)
Large hero or CTA contextAdd .btn-lg to any variant
